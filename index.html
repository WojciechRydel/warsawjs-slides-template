Isomorphic or Universal - what does it mean?

Isomorphic JS

Process or output is the same, but different implementation - e.g. Underscore and Lodash

Universal JS

The same code run both client and server side.

Classic JS client side app

Advanteges

  • Whole app lives in the client
  • Independed from server side

Disadavnteges

  • Cannot serve HTML to crawlers
  • Low performance of initial HTML rendering

Universal JS app

Advanteges

  • Better performance
  • SEO by default

Disadavnteges

  • Additional logic in the server side
  • Need to use JavaScript on the server side
